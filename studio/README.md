# Sanity Studio for Sibuk Bersantai

This folder contains a Sanity Studio configured for project `diz7kk6p` and dataset `production`.

## Usage

1. Install dependencies:
   ```bash
   cd studio
   npm install
   ```

2. Start Sanity Studio:
   ```bash
   npm run dev
   ```

3. Open the URL shown in the terminal.

## Notes

- The project ID is set to `diz7kk6p`.
- The dataset is set to `production`.
- CORS origin must include `https://sibukbersantai.github.io`.

## Schemas

- `siteSettings`: site title, subtitle, intro text, contact email, instagram URL
- `artwork`: title, description, url
